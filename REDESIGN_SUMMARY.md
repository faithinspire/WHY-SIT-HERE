## How to Use

### View in Browser
```
1. Open book-final.html in any web browser
2. Scroll through pages
3. Use browser zoom to adjust size
```

### Export to PDF
```
1. Open book-final.html in browser
2. Press Ctrl+P (Windows) or Cmd+P (Mac)
3. Select "Save as PDF"
4. Choose location and save
```

### Print Professionally
```
1. Export to PDF
2. Send to professional printer
3. Specify: 8.5" x 11", 20lb white bond, portrait
```
