# Quick Start Guide - Why Sit Here and Die?

## 🚀 Get Started in 30 Seconds

### Option 1: Read Online (Recommended)
1. **Open**: https://why-sit-here.vercel.app/app.html
2. **Click**: "GET BOOK" button
3. **Select**: Any chapter from the list
4. **Read**: Full-screen, click-to-read interface
5. **Navigate**: Use Previous/Next arrows or keyboard arrows (← →)

### Option 2: Download PDF
1. **Open**: https://why-sit-here.vercel.app/pdf-generator.html
2. **Click**: "📥 Generate PDF" button
3. **Download**: `Why_Sit_Here_and_Die.pdf` (14pt font, professional design)
4. **Read**: In any PDF reader

### Option 3: Local Testing
```bash
# Clone the repository
git clone https://github.com/faithinspire/WHY-SIT-HERE.git
cd WHY-SIT-HERE

# Open in browser
# Windows: start app.html
# Mac: open app.html
# Linux: xdg-open app.html

# Or use a local server
python -m http.server 8000
# Visit: http://localhost:8000/app.html
```
