## 🚀 HOW TO USE

### Method 1: Direct Read Online (Fastest)
1. Go to https://why-sit-here.vercel.app
2. Click **"READ ONLINE"** button in hero section
3. Full book opens immediately with all 18 chapters
4. Navigate chapters using sidebar
5. Scroll to read full content

### Method 2: Access Form (Collects Info)
1. Scroll to "Get the Book" section
2. Fill in: Name, Email, Reference
3. Click **"Access My Book Now"**
4. Access panel appears with:
   - **READ ONLINE** button
   - **DOWNLOAD PDF** button

### Method 3: Download PDF
1. After accessing book (either method)
2. Click **"DOWNLOAD PDF"** button
3. Print dialog opens
4. Select "Save as PDF"
5. Choose location and save
