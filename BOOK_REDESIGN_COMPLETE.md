## 📥 How to Use

### View in Browser
1. Open `book-final.html` in any web browser
2. Use browser zoom to adjust size
3. Print to PDF for distribution

### Print to PDF
1. Open `book-final.html` in browser
2. Press Ctrl+P (or Cmd+P on Mac)
3. Select "Save as PDF"
4. Configure print settings:
   - Margins: Default
   - Paper size: Letter (8.5" x 11")
   - Orientation: Portrait
5. Save and distribute

### Professional Printing
1. Export as PDF from browser
2. Send to professional printer
3. Specify:
   - Paper: 20lb white bond
   - Binding: Perfect bound or spiral
   - Trim size: 8.5" x 11"
