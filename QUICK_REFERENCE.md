## 🔧 Common Tasks

### Change Title
Find in HTML:
```html
<h1>Why Sit Here and Die?</h1>
```

### Change Author
Find in HTML:
```html
<strong>OLUSHOLA O. PAUL</strong>
```

### Change Font Size
Find in CSS:
```css
p { font-size: 1rem; }
```
Change `1rem` to desired size

### Change Margins
Find in CSS:
```css
.page { padding: 1in; }
```
Change `1in` to desired margin

### Add New Chapter
Copy this template:
```html
<div class="page page-content">
    <div class="page-content-inner">
        <div class="chapter-number">CHAPTER X</div>
        <h2 class="chapter-title">Title</h2>
        <p>Content here...</p>
    </div>
    <div class="page-number">XX</div>
</div>
```
