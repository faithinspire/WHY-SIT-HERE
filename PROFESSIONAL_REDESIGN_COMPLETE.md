### CSS Variables (Professional Standards)
```css

--page-height: 210mm (A5 height)
--margin-top: 20mm
--margin-bottom: 20mm
--margin-left: 18mm
--margin-right: 16mm
--line-height: 1.8
```

### Responsive Breakpoints
- **Desktop:** Full A5 dimensions
- **Tablet (≤768px):** 100% width, 15mm margins
- **Mobile (≤480px):** 100% width, 12mm margins

### Print Media Query
```css
@media print {
  @page { size: A5; margin: 0; }
  * { print-color-adjust: exact !important; }
  .page { page-break-after: always; }
}
```
