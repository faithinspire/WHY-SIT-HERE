# Quick Start — Apply Professional Layout in 5 Minutes

## TL;DR

Your book layout has been professionally redesigned. Here's how to apply it:

## OPTION 1: Copy-Paste CSS (Fastest)

1. Open `book.html` in a text editor
2. Find the `<style>` section in the `<head>`
3. Copy ALL content from `book-styles-optimized.css`
4. Paste it into the `<style>` section
5. Save the file
6. Open in browser and test

**Time: 2 minutes**

## OPTION 2: Link External File (Cleaner)

1. Save `book-styles-optimized.css` in your project folder
2. Open `book.html`
3. Find the `<head>` section
4. Add this line:
   ```html
   <link rel="stylesheet" href="book-styles-optimized.css">
   ```
5. Delete or comment out the old `<style>` section
6. Save and test

**Time: 3 minutes**

## VERIFY IT WORKS

1. Open `book.html` in your browser
2. Check:
   - ✅ All text fits on pages
   - ✅ Margins look even
   - ✅ Chapter titles are complete
   - ✅ No text extends beyond edges
3. Press Ctrl+P (or Cmd+P)
4. Select "Save as PDF"
5. Check PDF looks correct

**Time: 2 minutes**

## COMMON QUESTIONS

**Q: Will this break my current layout?**
A: No. The new CSS is designed to replace the old one completely. Just backup your file first.

**Q: Can I customize colors?**
A: Yes. Edit the `:root` variables in the CSS. See IMPLEMENTATION_GUIDE.md.

**Q: Does it work on mobile?**
A: Yes. The design is fully responsive.

**Q: Can I print it?**
A: Yes. Press Ctrl+P, select "Save as PDF", and it will be print-ready.
